
# Devkaar Online Store Dashboard

## Project Overview

The **Devkaar Online Store Dashboard** is a comprehensive analytics dashboard created using **Power BI Desktop**. It provides insightful metrics on orders, profit, and payment methods, helping businesses understand and improve their online sales performance.

## Key Features

- **Orders by City**: Visualizes order distribution across major cities, including Indore, Mumbai, Chennai, and more.
- **Profit by Subcategory**: Detailed profit breakdown by product subcategories like Bookcases, Printers, Tables, and Trousers.
- **Quantity by Category**: Shows product demand in key categories, such as Clothing, Furniture, and Electronics.
- **Payment Modes**: Analysis of payment methods used by customers, including Credit Card, COD, Debit Card, EMI, and UPI.
- **Total Sales and Profit**: Tracks the total amount, quantity, profit, and Average Order Value (AOV).

## Technology Stack

- **Power BI Desktop**: For creating interactive and insightful visualizations.
- **Data Source**: Sales and order data from the Devkaar Online Store.

## Insights Provided

This dashboard enables Devkaar Online Store to:
1. Analyze order patterns across cities to identify high-performing regions.
2. Understand profitability across various product subcategories.
3. Evaluate customer payment preferences for improved transaction strategies.
4. Track key metrics like total orders, profit, and AOV to drive data-informed decisions.

## Visualizations Included

1. **Orders by City**: Pinpoints city-specific order volumes.
2. **Profit by Subcategory**: Highlights profit contributions of product subcategories.
3. **Quantity by Product Category**: Shows demand distribution across categories.
4. **Payment Mode Analysis**: Provides insights into preferred payment options.
5. **Sales and Profit Summary**: Summarizes total quantity, profit, and AOV.

## Getting Started

### Prerequisites

- **Power BI Desktop** installed on your system.
- Access to the relevant sales dataset from Devkaar Online Store.

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/imuhsinalishah/devkaar-online-store-dashboard.git
   ```
2. Open the Power BI file in Power BI Desktop and connect it to your data.

## Usage

1. Load the dashboard in Power BI Desktop.
2. Interact with visualizations to explore sales data across different dimensions.
3. Apply filters to focus on specific metrics like city, product category, or payment mode.
